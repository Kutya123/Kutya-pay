# 🛒 Kutya Pay Store — Online Clothing Shop

Полноценный интернет-магазин одежды, созданный на HTML, CSS и JavaScript.

Проект реализует систему категорий, подкатегорий, динамическую загрузку товаров и корзину с подсчётом количества товаров.

## 🌍 О проекте

Kutya Pay Store — это онлайн-магазин с тремя основными категориями:

👔 Мужская одежда

👗 Женская одежда

🧒 Детская одежда

В каждой категории есть:

🔹 Одиночный товар

🔹 Наборы

Все товары динамически рендерятся через JavaScript.

# 🗂 Категории и подкатегории

# 🛍 Динамическая генерация товаров

Товары создаются через JS-функцию renderProductGroup()

Каждая карточка содержит:

Фото

Название

Цену

Кнопку "Добавить в корзину"
